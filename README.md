# Water Potability Classification

## ℹ️ About

A machine learning-driven approach to water potability classification based on physicochemical water quality metrics. This project focuses on developing high-accuracy predictive models to determine water safety for human consumption.

## ⚙️ Algorithm

- 🌳 **Random Forest**
- 📊 **Support Vector Machine**

## 🗂️ Dataset

The dataset utilized in this project is sourced from the [Water Quality and Potability](https://www.kaggle.com/datasets/uom190346a/water-quality-and-potability/data) dataset on Kaggle. It comprises various physicochemical properties of water samples, designed to support binary classification tasks.

## 🛠️ Technologies Used

- 🌐 **Programming Language:** Python.
- 🌱 **Environment:** Jupyter Notebook.
- ⚛️ **Libraries:** pandas, Matplotlib, seaborn, scikit-learn, and Joblib.
- ⚡ **Tool:** Google Colab.
